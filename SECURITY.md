# Security Policy

## Supported Scope

This repository contains skill definitions and metadata tooling, not application runtime code.

## Reporting a Vulnerability

If you discover a security issue, do not open a public issue.

1. Email: security@chalkpm.dev
2. Include: affected file(s), impact, reproduction, and suggested mitigation.
3. We will acknowledge receipt within 3 business days.

## Disclosure Process

- Validate and triage report.
- Coordinate fix with maintainers.
- Publish remediation details after fix is available.
